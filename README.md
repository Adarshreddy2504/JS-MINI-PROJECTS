# JavaScript Mini Projects

This repository contains the JavaScript mini projects I've built while learning frontend development. Each project focuses on a different concept such as DOM manipulation, events, Local Storage, APIs, form validation, timers, games, and UI interactions.

The goal of this repository is to practice JavaScript by building real projects instead of just solving coding exercises.

## 📂 Projects

| Project | Description | Skills / Concepts |
|---------|-------------|-------------------|
| 🎨 Color Palette Generator | Generate and explore beautiful color palettes. | DOM Manipulation, CSS, JavaScript |
| 💸 Expense Tracker | Track income, expenses, and overall balance. | Arrays, Local Storage, DOM |
| 🔖 Bookmark Saver | Save and manage website bookmarks. | CRUD Operations, Local Storage |
| 🍽️ Recipe Finder | Search recipes using an external API. | Fetch API, Async/Await, JSON |
| 👤 GitHub Finder | Search GitHub users and display profile details. | REST API, Fetch API, DOM |
| 💱 Currency Converter | Convert currencies using exchange rates. | API Integration, Events |
| 📝 To-Do App | Create, update, and delete daily tasks. | CRUD, Local Storage |
| ❌ Tic Tac Toe | Classic two-player Tic Tac Toe game. | Game Logic, Arrays |
| 🧠 Quiz Game | Interactive quiz with score tracking. | Arrays, DOM, Events |
| 🔐 Password Generator | Generate secure random passwords. | Strings, Randomization |
| 📋 Form Validator | Validate user inputs before submission. | Forms, Regex, Validation |
| 💰 Income Tax Calculator | Calculate tax based on user inputs. | Conditional Logic |
| 🎨 Background Changer | Change webpage backgrounds dynamically. | DOM Events, CSS |
| 📞 Contact Form | Responsive contact form with validation. | Forms, HTML, CSS |
| 📄 Contact Page | Responsive contact page UI. | HTML, CSS |
| ⏳ Scroll Progress | Display page scroll progress dynamically. | Scroll Events, DOM |
| 🚀 Coming Soon UI | Responsive "Coming Soon" landing page. | HTML, CSS |
| 👥 Players Showcase | Showcase player cards with responsive layouts. | CSS Grid, Flexbox |
| 💳 Pricing Cards | Responsive pricing cards component. | Flexbox, Responsive Design |
| 📝 Signup Design | Modern signup page UI design. | HTML, CSS, Responsive Design |

More projects will be added as I continue learning.

## Running the Projects

Clone the repository:

```bash
git clone https://github.com/Adarshreddy2504/JS-MINI-PROJECTS.git
```

Open any project folder and run the `index.html` file in your browser.
